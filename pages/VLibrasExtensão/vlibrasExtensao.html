Aqui você entenderá como conseguirá ler os livros desse site com o VLibras!

Instalação da Ferramenta

Para conseguir usar o tradutor, será necessário ter a ferramenta VLibras no seu navegador. É bem simples! Vamos lá?

Google Chrome

Visualizando a ferramenta

Clique aqui para visualizar a extensão VLibras.

Instalando a ferramenta

Clique em Usar no Chrome para instalar a extensão VLibras no seu navegador.

Usando a ferramenta

Em qualquer PDF, selecione o texto que deseja traduzir:

Clique com o botão direito do mouse e vá em Traduzir "..." Para LIBRAS:
